Submodules [hide private]
  • madgraph.core
    • madgraph.core.base_objects: Definitions of all basic objects used in the core code: particle, interaction, model, leg, vertex, process, ...
    • madgraph.core.color_algebra: Classes and methods required for all calculations related to SU(N) color algebra.
    • madgraph.core.color_amp: Classes, methods and functions required to write QCD color information for a diagram and build a color basis, and to square a QCD color string for squared diagrams and interference terms.
    • madgraph.core.diagram_generation: Classes for diagram generation.
    • madgraph.core.drawing: All the routines to choose the position to each vertex and the direction for particles.
    • madgraph.core.helas_objects: Definitions of objects used to generate language-independent Helas calls: HelasWavefunction, HelasAmplitude, HelasDiagram for the generation of wavefunctions and amplitudes, HelasMatrixElement and HelasMultiProcess for generation of complete matrix elements for single and multiple processes; and HelasModel, which is the language-independent base class for the language-specific classes for writing Helas calls, found in the iolibs directory
  • madgraph.interface
    • madgraph.interface.cmd_interface
    • madgraph.interface.extended_cmd: A file containing different extension of the cmd basic python library
    • madgraph.interface.launch_ext_program
    • madgraph.interface.tutorial_text
  • madgraph.iolibs
    • madgraph.iolibs.drawing_eps: This files contains class for creating files or object representing a diagram or a set of diagrams.
    • madgraph.iolibs.export_cpp: Methods and classes to export models and matrix elements to Pythia 8 and C++ Standalone format.
    • madgraph.iolibs.export_python: Methods and classes to export matrix elements to Python format.
    • madgraph.iolibs.export_v4: Methods and classes to export matrix elements to v4 format.
    • madgraph.iolibs.file_writers: Classes to write good-looking output in different languages: Fortran, C++, etc.
    • madgraph.iolibs.files: Methods and classes dealing with file access.
    • madgraph.iolibs.gen_infohtml
    • madgraph.iolibs.group_subprocs: Methods and classes to group subprocesses according to initial states, and produce the corresponding grouped subprocess directories.
    • madgraph.iolibs.helas_call_writers: Classes for writing Helas calls.
    • madgraph.iolibs.import_v4
    • madgraph.iolibs.misc: A set of functions performing routine administrative I/O tasks.
    • madgraph.iolibs.save_load_object: Function to save any Python object to file.
    • madgraph.iolibs.save_model: Function to save model files.
    • madgraph.iolibs.template_files
      • madgraph.iolibs.template_files.mg4_proc_card
    • madgraph.iolibs.ufo_expression_parsers: Parsers for algebraic expressions coming from UFO, outputting into different languages/frameworks (Fortran and Pythia8).
  • madgraph.various
    • madgraph.various.diagram_symmetry: Module for calculation of symmetries between diagrams, by evaluating amp2 values for permutations of modementa.
    • madgraph.various.process_checks: Several different checks for processes (and hence models): permutation tests, gauge invariance tests, lorentz invariance tests.
